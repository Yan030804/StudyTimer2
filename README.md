# 学习计时器

Windows 桌面学习时间记录工具，提供秒表、置顶紧凑窗口、历史明细编辑，以及七天、本月每周和本年每月统计图表。

## 使用方式

1. 双击 `StudyTimer.exe`。
2. 点击“开始学习”开始计时，可随时暂停或继续。
3. 点击“停止并保存”后，本次学习时长会写入当天文本文件。
4. 点击“紧凑模式”可切换为置顶悬浮窗口，拖动窗口空白位置可移动。
5. 在“历史记录”页可以新增、修改或删除当天明细。

## 数据位置

数据默认保存在：

```text
文档\StudyTimerData\年份\月份\日期.txt
```

例如：

```text
文档\StudyTimerData\2026\06\2026-06-15.txt
```

每日文件包含总学习时长、总秒数和每次学习的开始与结束时间。一次学习跨过午夜时，会自动拆分到两个日期文件。

计时过程中每 5 秒更新一次 `.state.json` 恢复文件。程序异常退出后，下次启动会询问是否恢复到最后一次可靠时间。

## 开发与发布

项目使用 C#、WPF 和 .NET 8，不依赖第三方 NuGet 包。运行以下脚本可生成 Windows x64 自包含单文件程序：

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\publish.ps1
```

输出文件：

```text
dist\win-x64\StudyTimer.exe
```
